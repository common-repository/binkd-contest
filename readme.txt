(Payment Required) Gives the ability to embed a contest from the Binkd Promotion Platform in your Pages or Posts.

== Description ==

You can create a contest on the <a href="https://promotion.binkd.com/Account/Default.aspx">Binkd Promotion Platform</a> and embed it on any page or post on your site.

Contest types available:

*   Word Contests
*   Twitter Contests
*   Photo Contests
*   Sweepstakes
*   Challenge Contests
*   Video Contests


Any upgrade or new contest types to the Binkd Promotion Platform will automatically be available to you and there will be no need to upgrade this plug-in.

Contests are a great way to increase your following and engagement to your site.

== Installation ==

This section describes how to install the plug-in and get it working.

1. Upload `binkd-contest` to the `/wp-content/plugins/` directory
2. Activate the plug-in through the 'Plug-ins' menu in WordPress
3. Go to Binkd Contest on the Settings menu to enter your API Key

== Frequently Asked Questions ==

= Are contests hosted on my site? =

No, contests are hosted via the Binkd Promotion Platform. This plug-in gives you the ability to seamlessly integrate it with your WordPress installation.

= Do I need a Binkd account? =

Yes, you need to signup at https://promotion.binkd.com/Account/Register.aspx in order to use this plug-in.

= Is this free? =

Twitter contests are free, however all other contests require payment. Our payment schedule can be found at http://www.binkd.com/pricing/
